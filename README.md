# C# SDK 使用说明 - 字库系列

## 1. 平台支持

本项目由纯C++编写， 因此，其可以正常运行于Windows, Linux平台。本文档主要针对动态链接库（C++）开发的相关说明，需要协议开发请使用我们提供的[协议文档](http://judy2gong.gitee.io/docblog/zh/k/potocol/)进行开发。
如果你想尽快开发出一个能简单控制的软件，建议按以下流程阅读本文档后进行软件开发。



- 获取SDK
- 阅读字库C++动态库接口使用说明
